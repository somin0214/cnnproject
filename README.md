# cnnproject
1. 제목 : MNIST를 활용한 CNN 프로젝트 

2. 프로젝트 수행 날짜 : 2023년 1월 ~ 2월()

3. 기간 : 알파코 딥러닝 부트캠프 7기

4. 데이터 : 0~9의 숫자 이미지로 이루어진 MNIST train 6만장 test 3만장으로 사용

5. 역할 : CNN 모델 개선, 데이터 정제 및 전처리, 발표, 팀장

6. Method : 웹과 딥러닝
